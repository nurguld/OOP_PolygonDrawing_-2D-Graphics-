# SWE 104 Object-Oriented Programming - Polygon Drawing Project

**Course:** SWE 104 - Object-Oriented Programming  
**Semester:** Spring 2024-2025  


---

## 📌Project Description
This Windows Forms application allows users to create, draw, and rotate regular polygons. 
The application includes the following classes:

- **Point2D**: Represents a 2D point with Cartesian (x, y) and Polar (r, θ) coordinates.
- **ColorRGB**: Stores RGB color values for the polygons.
- **Polygon**: Defines a regular polygon with properties like center, length, number of edges, and color.

The form interface allows users to set polygon parameters, draw the polygon on a PictureBox, display vertex coordinates, and rotate the polygon clockwise or counter-clockwise.

---

## 📌 Features
- Create polygons with custom center coordinates, length, number of edges, and color.
- Display edge coordinates in a ListBox.
- Rotate polygons clockwise or counter-clockwise by entering a rotation angle.
- Reset all polygon parameters to random values.
- Real-time drawing and updating of polygons on the PictureBox.
